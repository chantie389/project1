APP REPORT



PURPOSE OF THE APPLICATION

this application was developed to help users maintain social connections during busy days. 
It was inspired by a close friend who struggles to stay in touch regularly.


DESIGN CONSIDERATIONS

The app was designed to be simple, user-friendly, and engaging. it uses a time-based structure
(morning,afternoon,evening) to match real-life routines and ensure ease of use. the interface
is clear and responsive to provide a smooth user experience.

USE OF GitHub 

GitHub was used for version control, allowing changes to be tracked and the project to be 
securely stored and managed

USE OF GitHub ACTIONS

GitHub Actions was used to automate tasks such as building and testing the application,
improving development efficiency (OpenAI,2026).
